# 🚀 Bootstrap Multi-Page Website Project

This project is a responsive multi-page website developed using **HTML** and **Bootstrap 5** as part of my learning journey and internship at **MMA Technology**.

---

## 📌 Project Overview

The project consists of three interconnected web pages:
- 🏠 Home Page (Bootstrap Grid & Progress Bars)
- 📝 Admission Form Page
- 👨‍💻 About Me Page

It demonstrates the use of modern frontend practices including responsive layouts, navigation systems, and UI components.

---

## ✨ Features

- ✅ Responsive Navigation Bar  
- ✅ Multi-page Linking (Home, Admission, About Us)  
- ✅ Bootstrap Grid System  
- ✅ Progress Bars for Skills  
- ✅ Styled Admission Form  
- ✅ Modern UI Design with Bootstrap & Icons  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  
- Bootstrap Icons
